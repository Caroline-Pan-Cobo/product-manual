Set up your Exchange Wallets on Cobo Portal by linking your exchange accounts using these simple steps:

1) Log in to Cobo Portal.
2) Click <img src="/images/exchange-wallet/1.png" className="icon"></img> and then select **Exchanges**.
3) Click **+ Exchange Account**, or alternatively, click **Link Account** under the specific exchange.
4) On the exchangeâ€™s platform, whitelist Cobo IP (**18.177.176.209**) in your API key settings and generate an API key and API secret. Refer to the next pages for detailed instructions for the respective exchanges.
5) Fill in the required information (name, API key, secret, and passphrase) and click **Next**.
6) (Optional) Select the exchange sub-accounts you wish to link, if applicable.
7) Click **Link** to complete the setup. Once successful, you will see the linked exchange accounts in your Exchange Wallets dashboard.

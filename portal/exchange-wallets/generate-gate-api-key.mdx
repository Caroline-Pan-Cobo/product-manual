Generate the **API Key** and **API Secret** key for your Gate account by following these steps:

1) Log into your Gate account.
2) Navigate to **My Profile** > **API Management**.
3) Select **Create API Key**.
4) Fill in API Key Remark = **Cobo**.
5) Fill in Coboâ€™s IP address under IP Permissions = **18.177.176.209**.
6) Select **Bind IP**.
7) Select **API v4 Key**.
8) Select **Trading Account**.
9) Toggle on the following **5 Permissions**:<br />
    a) Spot Trade: Read-Only<br />
    b) Wallet: Read And Write<br />
    c) Withdraw: Read And Write<br />
    d) Subaccount: Read-Only<br />
    e) Account: Read-Only
10) Click **Confirm**.

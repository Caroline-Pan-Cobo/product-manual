Generate the **API Key** and **API Secret** key for your Bybit account by following these steps:

1) Log into your Bybit account.
2) Navigate to **My Profile** > **Account & Security** > **API Management**.
3) Select **Create New Key**.
4) Select **System-generated API Keys**.
5) Select **API Transaction**.
6) Fill in name for the API key = **Cobo**.
7) Select the following **2 API Key Permissions**:<br />
    a) Read-Write<br />
    b) Only IPs with permissions granted are allowed to access the OpenAPI<br />
8) Fill in Coboâ€™s IP address under API Key Permissions = **18.177.176.209**.
9) Tick the following **4 options under "Read-Write"**:<br />
    a) Account Transfer<br />
    b) Subaccount Transfer<br />
    c) Withdrawal<br />
    d) Exchange History<br />
10) Click **Submit**.

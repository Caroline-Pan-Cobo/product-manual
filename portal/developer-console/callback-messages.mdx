Manage the callback messages in your Developer Console easily by clicking on the corresponding icons:

- **Search**: Quickly locate your callback by searching with the URL, message, or message ID.
- **Tick Box**: Select callback message display based on the status of the message.
- **Time**: Select a timeframe to display all callback messages within that time period.
- **Message Types**: Tick the type of message from the drop-down menu.
- **Clear All**: Remove all selected conditions.
- **Retry**: When the status is either "Failed" or "Retrying", the user can resubmit the message.

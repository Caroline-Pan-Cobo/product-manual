Manage the API logs in your Developer Console easily by clicking on the corresponding icons:

- **Search**: Quickly locate your API logs by searching with the API key, IP address, request ID, or API endpoint.
- **Tick Box**: Select API log display based on the status of the API request.
- **Time**: Select a timeframe to display all API requests within that time period.
- **Method**: Tick the type of API request method from the drop-down menu.
- **More**: Enter the API key, API endpoint, IP address, status code, or API request time to find the desired log details.
- **Clear All**: Remove all selected conditions.

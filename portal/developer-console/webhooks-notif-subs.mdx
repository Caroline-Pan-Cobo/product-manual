title: "Available event notification subscriptions"
sidebarTitle: "Available event notification subscriptions"
---

Create a webhook endpoint with the following event notifications:

- **Added Account Update**: Receive notifications for new account creation.
- **Vault Account Added**: Receive notifications for new vault creation.
- **Transaction Status Update**: Receive notifications for updates in transaction status.
- **Transaction Created**: Receive notifications for new transaction creation.
- **New Withdraw Request**: Receive notifications for requests in withdrawals.
- **Speed Up Transaction**: Receive notifications for requests in speeding up transactions.
- **Retry Double Check**: Receive notifications for confirmation prompts.

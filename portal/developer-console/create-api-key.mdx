Create an API key following these simple steps:

1. Log into Cobo Portal.
2. On the left navigation menu, click **Developer Console** > **API Keys**.
3. Click **Create an API key**.
4. Enter the API key name (maximum 30 characters).
5. Enter the external API key (maximum 64 characters, containing only numbers 0 to 9 and alphabets A to F).
6. Select the user role from the drop-down list (able to select more than one).
7. Tick either one of the two key types.
8. Enter the IP address to whitelist (separate between each IP address with a comma, if you have more than one IP address to whitelist).
9. Click **Confirm**.

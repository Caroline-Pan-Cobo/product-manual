In addition to the preset user roles Cobo provides, you can create your own customized user roles.

Create a new user role by clicking **Organization** > **Security** > **Roles** > **Create New User Roles**.

**Note**: For the App setting on the Create New Role dialog, select an app this user role will apply to.

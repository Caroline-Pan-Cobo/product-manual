A vault is a collection of wallets sharing the same root extended private key. Follow the steps below to create a vault.

**Note:** Only the users who have been assigned [the Operator user role in the organization](/portal/organization/roles-and-permissions) can create a vault.

1. On Cobo Portal, click <img src="/images/mpc-wallet/wallets-icon.svg" className="icon"></img> > **MPC Wallets** > **Create Vault**.

<img src="/images/mpc-wallet/create-vault.png" className="screenshot_full_screen"/>

2. Name your vault, then click **Create**.

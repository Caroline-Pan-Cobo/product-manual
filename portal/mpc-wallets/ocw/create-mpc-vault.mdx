title: "Create an MPC vault"
sidebarTitle: "Create an MPC vault"
---

A vault is a collection of wallets sharing the same root extended private key. Follow the steps below to create a vault.

**Note:** Only the users who have been assigned [the Operator user role in the organization](/cobo-portal/organization/security/overview1) can create a vault.

1. On Cobo Portal, click the Wallet icon, then click **MPC** > **Create Vault**.

2. Name your vault, then click **Create**.

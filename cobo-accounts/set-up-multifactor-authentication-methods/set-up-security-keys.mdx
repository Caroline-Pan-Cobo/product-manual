---
title: "Set up security keys"
---
_A security key is a secure physical device used to assist in verifying your identity during MFA verification. You can add up to 10 security keys to your Cobo Accounts._


1) Log in to Cobo Accounts and click on **MFA Management** in the main menu on the left hand side.
2) Click on **Add** next to Security Key.
3) To successfully add a security key, you must insert the device and touch it.
4) You can review the details of your newly added security key, including its creation time and the last time it was used.

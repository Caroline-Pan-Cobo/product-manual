You can assign user roles to either new or existing members.

## Assign user roles to new members

While inviting new members, you can assign user roles to them under **User Role Settings**.

## Assign user roles to existing members

You can assign user roles to existing members by clicking **Organization** > **Members** > **Modify User Role**.

In addition to the preset user roles provided by Cobo, you can also create your own customized user roles.

To create a new user role, please click **Organization** > **User Roles** > **Create User Role**.

<img src="/en/images/organization/org9.png" className="screenshot_modal"/>

**Note**: In the **Create New User Role** dialog, please specify whether this user role will apply to web operations, API operations, or both.

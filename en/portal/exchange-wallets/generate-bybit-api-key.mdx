Generate the **API Key** and **API Secret** for your Bybit account by following these steps:

1) Log in to your Bybit account.
2) Navigate to **My Profile** > **Account & Security** > **API Management**.
3) Select **Create New Key**.
4) Select **System-generated API Keys**.
5) Select **API Transaction**.

<img src="/en/images/exchange-wallet/7.png" className="screenshot_modal"/>

6) Fill in the name for the API key = **Cobo**.
7) Select the following **2 API Key Permissions**:<br />
    a) Read-Write<br />
    b) Only IPs with permissions granted are allowed to access the OpenAPI<br />
8) Fill in Coboâ€™s IP address under API Key Permissions = **18.177.176.209**.
9) Tick the following **4 options under "Read-Write"**:<br />
    a) Account Transfer<br />
    b) Subaccount Transfer<br />
    c) Withdrawal<br />
    d) Exchange History<br />
<img src="/en/images/exchange-wallet/8.png" className="screenshot_modal"/>

10) Click **Submit**.

You have now generated the **API Key** and **API Secret** and can now return to Cobo Portal interface to complete the set-up. For more information, please refer to this [guide](https://www.bybit.com/en/help-center/article/How-to-create-your-API-key).

Generate the **API Key** and **API Secret Key** for your Bitget account by following these steps:

1) Log in to your Bitget account.
2) Navigate to **Personal Center** > **API Management**.
<img src="/en/images/exchange-wallet/bitget api.png" className="screenshot_modal"/>

3) Click **Create New Key**.
4) Fill in all required fields. Please note that your passphrase will be required when connecting the exchange to Cobo Portal.
5) Select Permissions = **Read-write**.
6) Select Spot = **Trade**.
7) Select Wallet = **Withdraw**.
8) Fill in Coboâ€™s IP address under IP whitelist = **18.177.176.209**.
9) Click **Next**.

You have now generated the **API Key** and **API Secret Key** and can now return to Cobo Portal interface to complete the set-up. For more information, please refer to this [guide](https://www.bitget.com/academy/bitget-api-trading-guideline).

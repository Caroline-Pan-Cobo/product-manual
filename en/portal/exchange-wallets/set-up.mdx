This article describes how to set up your Exchange Wallets on Cobo Portal by linking your exchange accounts.

## Prerequisites

- Set up your Cobo Account as described in [Quick start guide](/en/portal/quick-start-guide).
- Generate and set up the API key on the exchange that you use as described in the following guides:
    - [Binance](generate-binance-api-key.mdx)
    - [Bitget](generate-bitget-api-key.mdx)
    - [Bybit](generate-bybit-api-key.mdx)
    - [Deribit](generate-deribit-api-key.mdx)
    - [Gate.io](generate-gate-api-key.mdx)
    - [OKX](generate-okx-api-key.mdx)

## Steps
1. Log in to [Cobo Portal](https://portal.cobo.com/login).
2. Click the wallet icon <img src="/en/images/exchange-wallet/1.png" className="icon"></img> and then select **Exchange Wallets**.
3. Click the **+ Exchange Account** button.
4. On the **Select Exchange** dialog, select the exchange that you use and click **Continue**.
5. The required information vary by exchange. In most cases you need to select the account type and provide a wallet name, an API key,and an API secret.
    <Info>When linking a Main Account, you can link its Sub Accounts at the same time.</Info>
    <Note>When linking a Sub Account, you need to link its Main Account first, and select the Main Account as the **Associated Main Account**.</Note>
    <img src="/en/images/exchange-wallet/link-dialog.png" className="screenshot_modal" alt="The dialog of linking exchange account" />
6. Click **Link** to complete the setup. 

Once your exchange account is linked, it will display in your Exchange Wallets dashboard.

<img src="/en/images/exchange-wallet/linked-accounts.png" className="screenshot_full_screen" alt="The Exchange Wallets dashboard"/>

## Next-steps

When you set up your Exchange Wallets on Cobo Portal, you can perform actions on the Exchange Wallets, or transfer assets from or to your Exchange Wallets. For more details, see [Manage Exchange Wallets](manage) and [Transfers](transfers).

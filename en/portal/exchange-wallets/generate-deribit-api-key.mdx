Generate the **Client ID** and **Client Secret** for your Deribit account by following these steps:

1) Log in to your Deribit account.
2) Navigate to **My Account > API**.
3) Select **+ ADD NEW KEY**.

<img src="/en/images/exchange-wallet/deribit.png" className="screenshot_full_screen"/>

4) Select Account = **read**.
5) Select Wallet = **read_write**.
6) Fill in Coboâ€™s IP address under IP whitelist = **18.177.176.209**.
7) Click **Create a new API key**.
8) After the API key is created, click **Load Keys**.

You have now generated the **Client ID** and **Client Secret** and can now return to Cobo Portal interface to complete the set-up.

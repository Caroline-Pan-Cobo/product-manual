Generate the **API Key** and **API Secret** for your OKX account by following these steps:

1) Log in to your OKX account.
2) Navigate to **Exchange** > **My Account** > **API**.
3) Select **+ Create API key**.
4) Fill in API name = **Cobo**.
5) Fill in Passphrase = **Enter a secure passphrase of your choice**.
6) Fill in Coboâ€™s IP address under the IP address allowlist =  **18.177.176.209**.
7) Select **API trading**.
8) Select the following **3 Permissions**:<br />
    a) Read<br />
    b) Withdraw<br />
    c) Trade
<img src="/en/images/exchange-wallet/5.png" className="screenshot_modal"/>

9) Click **Confirm**.

You have now generated the **API Key** and **API Secret** and can now return to Cobo Portal interface to complete the set-up.

When you link an OKX account to your Exchange Wallet, you will need to provide an API key, an API secret, and a passphrase. Besides, you also need to include Cobo in the IP whitelist and assign specific permissions to the key.

This article explains how to generate an API key on OKX, including adding Cobo to the IP whitelist and setting permissions.

<Note>The content is subject to changes on OKX. For more details, refer to <a href="https://www.okx.com/docs-v5/en/#overview-v5-api-key-creation">Generating an API Key</a>.</Note>

1. Log in to your OKX account.
2. Navigate to **Exchange** > **My Account** > **APIs**.
3. Click **+ Create APIs** to open a **Create APIs** page.
    <img src="/en/images/exchange-wallet/okx.png" className="screenshot_modal" alt="The Create APIs page" />
4. Enter an **API name** in the field.
5. In the **Account** section, select the OKX account for which you create the API key.
6. In the **Purpose** section, select **API trading**.
7. In the **IP address allowlist** field, enter Cobo's IP address `18.177.234.7` and `18.180.16.190`.
8. Select the following **Permissions**:
    - **Read**
    - **Withdraw**
    - **Trade**
9. Enter a **Passphrase** in the field.
8. Click **Confirm** to generate the API key.

You have now generated and set up an API Key. You can go to Cobo Portal to link your OKX account with the API key, API secret, and passphrase. To learn how to link your account, see [Set up Exchange Wallets](/en/portal/exchange-wallets/set-up).

## WaaS 2.0
The WaaS 2.0 tab provides an in-depth view of API requests made by members of your organization to the WaaS 2.0 service, offering tools and insights to ensure secure, efficient, and responsive operations.

### Overview
Comprehensive overview that displays real-time metrics and visualizations of your organization's API Requests, API Request Distribution, Top 5 API Operations, Webhooks, and Callbacks. These insights provide valuable information about your users' usage patterns, enabling you to optimize resources and identify potential issues proactively.

### API keys
Manage your organization's API keys, which are essential for authenticating and authorizing requests to your organization on Cobo Portal. You can create, edit, revoke, and monitor the usage of your API keys on this page.

### Webhooks
Webhooks enable real-time event notifications. Configure and manage your webhook endpoints, ensuring that the desired organization users stay informed about critical events as they occur.

### Callbacks
Callbacks execute codes in response to a specific event. Configure and manage your callback endpoints, ensuring that specific user-initiated events will trigger a predetermined response.

### API logs
Comprehensive log viewer, providing detailed information about all the API communication requests with your organization. This feature is important for debugging, troubleshooting, and auditing purposes, ensuring that you have complete visibility into the interactions between your organization and users.

### Webhook events
Detailed view of all the webhook events that have occurred between Cobo Portal and your integrated application. This feature allows you to track and monitor specific events, enabling you to gain oversight of the status of all event notifications.

### Callback messages
Monitor the status of all triggered callback messages here. This feature is useful for debugging and ensuring that your organization is handling user-initiated responses correctly.

## Cobo Portal Apps
The **Cobo Portal Apps** tab lists all Cobo Portal Apps developed by members of your organization, highlighting API activity at the app level.
<Info>If no Cobo Portal Apps have been developed by members of your organization, this tab will not display.</Info>

### API Dashboard
The API Dashboard serves as a central hub for real-time visualizations and metrics related to API requests made by the Cobo Portal Apps developed by your organization's members. This dashboard offers valuable insights into usage patterns, enabling developers to optimize resources and proactively address potential issues.

### App Usage
This dedicated page allows you to monitor the usage metrics of the Cobo Portal Apps developed by members of your organization. It helps developers track performance and identify areas for improvement, facilitating efficient resource management.

### API Logs
Similar to the WaaS 2.0 API Logs, this section provides comprehensive information on all API communications specific to Cobo Portal Apps. It supports debugging, troubleshooting, and auditing, ensuring you have full visibility into app interactions.

### Webhook Events
This section provides a complete view of all webhook events associated with the Cobo Portal Apps developed by members of your organization. It allows you to track, monitor, and verify event notifications, helping maintain real-time awareness of all app activities.

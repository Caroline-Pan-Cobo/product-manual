---
title: "Why register API keys on Developer Console?"
sidebarTitle: "Why register API keys on Developer Console?"
---

Admins and Operators can register API keys on the Developer Console and assign specific user roles, permissions, and IP whitelists. Multiple API keys can be registered depending on the different usage scenarios.

- **Specify user roles and permissions**: Determine and assign specific user roles and permissions for the API key.
- **Specify IP whitelist**: Determine the IP addresses that are able to access the API keys.

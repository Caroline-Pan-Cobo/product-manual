Create a webhook endpoint following these simple steps:

1. Log in to [Cobo Portal](https://portal.cobo.com/login).
2. On the left navigation menu, click **Developer** > **Webhooks**.
3. Click **Create a Webhook Endpoint**.
4. Enter the URL link.
5. Enter a description for the webhook (maximum 30 characters).
6. Select the types of event notification you wish to receive (able to select more than one).
7. Click **Create**.

---
title: "Available callback message types"
sidebarTitle: "Available callback message types"
---

A callback message will be sent when one of the following occurs:

- **Exchange Wallet Withdrawal**: A user withdraws tokens from the Exchange Wallet.
- **Custodial Wallet Withdrawal**: A user withdraws tokens from the Asset/Web3 Wallet.
- **Smart Contract Wallet Withdrawal**: A user withdraws tokens from the Safe Wallet.
- **MPC Wallet Withdrawal**: A user withdraws tokens from the Organization-controlled / User-controlled Wallet.

---
title: "Available callback message types"
sidebarTitle: "Available callback message types"
---

A callback message will be sent when one of the following occurs:

- **Exchange Wallet Withdrawal**: A user withdraws tokens from an Exchange Wallet.
- **Custodial Wallet Withdrawal**: A user withdraws tokens from a Custodial Wallet.
- **Smart Contract Wallet Withdrawal**: A user withdraws tokens from a Smart Contract Wallet.
- **MPC Wallet Withdrawal**: A user withdraws tokens from an MPC Wallet.

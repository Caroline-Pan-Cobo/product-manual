Manage the API logs in your Developer Console easily by clicking on the corresponding icons:

- **Search** <img src="/en/images/developer-console/icon_search.svg" className="icon"></img>: Quickly locate your API logs by searching with the API key, IP address, request ID, or API endpoint.
- **Tick Boxes**: Select API log display based on the status of the API request.
- **Time** <img src="/en/images/developer-console/button_time.svg" className="icon"></img>: Select a timeframe to display all API requests within that time period.
- **Method** <img src="/en/images/developer-console/button_method.svg" className="icon"></img>: Tick the type of API request method from the drop-down menu.
- **More** <img src="/en/images/developer-console/button_more.svg" className="icon"></img>: Enter the API key, API endpoint, IP address, status code, or API request time to find the desired log details.
- **Clear All** <img src="/en/images/developer-console/button_clear all.svg" className="icon"></img>: Remove all selected conditions.

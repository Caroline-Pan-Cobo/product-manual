Create an API key following these simple steps:

1. Log in to [Cobo Portal](https://portal.cobo.com/login).
2. On the left navigation menu, click **Developer** > **API Keys**.
3. Click **Create API Key**.
4. Enter the API key name (maximum 30 characters).
5. Enter the API key (64 characters, containing only numbers 0 to 9 and alphabets A to F).
6. Select the user role from the drop-down list (able to select more than one).
7. Select the wallets this API key has control over.
8. Tick either one of the two key types.
9. Enter the IP address to whitelist (separate between each IP address with a comma, if you have more than one IP address to whitelist).
10. Click **Create**.

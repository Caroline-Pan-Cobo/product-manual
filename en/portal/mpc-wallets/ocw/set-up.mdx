Setting up an Organization-Controlled Wallet requires several steps:
1. [Create an MPC vault](/en/portal/mpc-wallets/ocw/create-mpc-vault)
2. [Create key share groups](/en/portal/mpc-wallets/ocw/create-key-share-groups)
3. [Back up key share groups](/en/portal/mpc-wallets/ocw/back-up-key-share-groups)
4. [Create an MPC wallet](/en/portal/mpc-wallets/ocw/create-mpc-wallet)

Follow the instructions in each topic to work through the steps described above.

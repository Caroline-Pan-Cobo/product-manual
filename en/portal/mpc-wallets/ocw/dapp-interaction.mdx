To enable seamless interaction with dApps using MPC Wallets, you must first install Cobo Connect, a Chrome extension that functions as the gateway between Cobo Portal and the dApp ecosystem. For more information about Cobo Connect, see the Cobo Connect [documentation](/en/cobo-connect/set-up).

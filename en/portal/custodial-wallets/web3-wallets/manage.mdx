1. Log in to [Cobo Portal](https://portal.cobo.com/login).
2. Click <img src="/en/images/custodial-wallet/wallet-icon.svg" className="icon"/> and select **Custodial Wallets**. Select the **Web3 Wallets** tab.
3. Manage your wallets easily by clicking the corresponding icons.

    <img src="/en/images/custodial-wallet/web3/web3-wallet-list.png" className="screenshot_full_screen"/>

    - Search <img src="/en/images/custodial-wallet/search.svg" className="icon"/>: Search for specific wallets by wallet name or address.
    - Change Views <img src="/en/images/mpc-wallet/icon_list.svg" className="icon"></img>: Switch between grid view and list view.

    In the row for each wallet:

        - View Transaction History <img src="/en/images/custodial-wallet/transaction-history.png" className="icon"/>: View the wallet’s transaction history.
        - Cobo Connect <img src="/en/images/common/icon_cobo_connect.svg" className="icon" alt="Cobo Connect"></img>: Launch Cobo Connect.
        - View Wallet Details <img src="/en/images/common/icon_view.svg" className="icon" alt="View details"></img>: View detailed information about the wallet.

4. Click a Web3 Wallet to view its details. 

    - On the **Addresses** tab:

        <img src="/en/images/custodial-wallet/web3/addresses-tab.png" className="screenshot_full_screen"/>

        - Search <img src="/en/images/custodial-wallet/search.svg" className="icon"/>: Search for specific addresses by address or address label.
        - Filter <img src="/en/images/custodial-wallet/filter.png" className="icon"/>: Filter the addresses by network and balance.
        - Sort <img src="/en/images/smart-contract-wallet/button_sort.svg" className="icon"></img>: Sort by balance, created time, or last activity time.
        - Create Address: Create an address.
        
        In each row:
            - Deposit <img src="/en/images/custodial-wallet/deposit-icon.png" className="icon"/> and Withdraw <img src="/en/images/custodial-wallet/withdraw-icon.png" className="icon"/>: For detailed instructions, see [Transfers (deposit / withdraw)](/en/portal/custodial-wallets/web3-wallets/transfers).
            - View Transaction History <img src="/en/images/custodial-wallet/transaction-history.png" className="icon"/>: View the address’s transaction history.
            - Cobo Connect <img src="/en/images/common/icon_cobo_connect.svg" className="icon" alt="Cobo Connect"></img>: Launch Cobo Connect.
            - View Details <img src="/en/images/common/icon_view.svg" className="icon" alt="View details"></img>: View detailed information about the address.
            - Edit Address Label <img src="/en/images/mpc-wallet/icon_edit.svg" className="icon"></img>: Modify the address labels.

    - On the **Tokens** tab:

        <img src="/en/images/custodial-wallet/web3/tokens-tab.png" className="screenshot_full_screen"/>

        - Search <img src="/en/images/custodial-wallet/search.svg" className="icon"/>: Search for specific tokens by token name or address.
        - Filter <img src="/en/images/custodial-wallet/filter.png" className="icon"/>: Filter the tokens by token name, last activity time, and balance.
        - Sort <img src="/en/images/smart-contract-wallet/button_sort.svg" className="icon"></img>: Sort by token name, balance, or last activity time.
        - Add Tokens: Add token types to your wallet.
        
        In each row:
            - View Deposit Addresses <img src="/en/images/custodial-wallet/view-deposit-address.png" className="icon"/>: View deposit addresses of the token.
            - Deposit <img src="/en/images/custodial-wallet/deposit-icon.png" className="icon"/> and Withdraw <img src="/en/images/custodial-wallet/withdraw-icon.png" className="icon"/>: For detailed instructions, see [Transfers (deposit / withdraw)](/en/portal/custodial-wallets/web3-wallets/transfers).
            - View Transaction History <img src="/en/images/custodial-wallet/transaction-history.png" className="icon"/>: View the token’s transaction history.

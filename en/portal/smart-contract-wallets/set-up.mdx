Now that you are a Cobo Argus user who has already set up Safe&#123;Wallet&#125; with Cobo Safe enabled. Follow the instructions below to set up a Smart Contract Wallet on Cobo Portal.

1. To set up a Smart Contract Wallet on Cobo Portal, click [Insert icon]  > **Smart Contracts** > **Import**.
2. Select Safe&#123;Wallet&#125;, then click **Next**.
3. Cobo Portal will start to import your Safe&#123;Wallet&#125; from Cobo Argus.
4. Select one or more Safe&#123;Wallet&#125; you want to import, then click **Import**.
5. Click **OK** when you see the Congratulations! dialog.
6. On Cobo Portal, confirm that your Smart Contract Wallet has been set up.

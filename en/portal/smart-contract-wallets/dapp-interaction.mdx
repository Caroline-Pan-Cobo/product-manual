To enable seamless interaction with dApps using Smart Contract Wallets, you must first install Cobo Connect, a Chrome extension that functions as the gateway between Cobo Portal and the dApp ecosystem. Click here to learn more about [Cobo Connect](/en/cobo-connect/interact-with-dapps).

To install Cobo Connect, click into your Safe&#123;Wallet&#125; on Cobo Portal and click the Cobo Connect icon in the upper right corner. For more information about Cobo Connect, see the Cobo Connect [documentation](/en/cobo-connect/set-up).

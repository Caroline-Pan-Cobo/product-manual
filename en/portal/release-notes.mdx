## June 10, 2024

This is the first release of [Cobo Portal](/en/portal/introduction), an all-in-one platform designed to provide a secure and efficient way to manage your digital assets. With this release, Cobo Portal offers the following features:

- [Custodial Wallets](/en/portal/custodial-wallets/introduction) and [MPC Wallets](/en/portal/mpc-wallets/introduction) combined in one platform.
- [Risk Controls](/en/portal/risk-controls/introduction), which offer comprehensive risk control mechanisms designed to address the challenges in storing, managing, and transferring digital assets, including off-chain transaction policies, governance policies, and user roles and permissions.

- [Developer Console](/en/portal/developer-console/introduction), which is a central hub for developers to manage API keys, track API logs, oversee webhook events, and handle callback messages.
- [Dashboard](/en/portal/dashboard/introduction), which provides a comprehensive overview of your total asset value, asset allocations, transactions, pricing plan usage, and the latest Cobo ecosystem updates.

To learn more about Cobo Portal, see [Introduction to Cobo Portal](/en/portal/introduction).

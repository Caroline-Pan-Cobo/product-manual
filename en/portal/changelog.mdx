<Update label="2024/12/11 - 2025/01/07">
**Cobo Portal Apps documentation**
- Added new Token Bridge app documentation with comprehensive guides on [setup](/en/apps/bridge/how-to-use/setup), [managing the dashboard](/en/apps/bridge/how-to-use/manage/dashboard), and [bridging assets](/en/apps/bridge/how-to-use/manage/bridge-form)
- Added new Screening app documentation covering [setup](/en/apps/screening/set-up), [managing recipients](/en/apps/screening/recipients), and [viewing screening results](/en/apps/screening/viewscreening)
- Added workflow documentation for Batch Payouts including Guard approval process in [Make a batch payout](/en/apps/batch-payouts/make-payout)
- Updated Superloop documentation with new supported exchanges: [Bitget](/en/apps/superloop/supported-exchanges-bitget) and [BitMart](/en/apps/superloop/supported-exchanges-bitmart)

**MPC Wallets documentation**
- Updated terminology from "API Co-Signer" to "Server Co-Signer" across all documentation
- Added comprehensive documentation about server co-signer in [Server co-signer](/en/portal/mpc-wallets/ocw/server-co-signer)
- Added embedded video tutorials to [Introduction to Organization-Controlled Wallets](/en/portal/mpc-wallets/ocw/introduction)
- Updated documentation for [key share verification](/en/portal/mpc-wallets/ocw/key-share-verification) and [recovery procedures](/en/portal/mpc-wallets/ocw/recover-root-key)

**User roles and permissions**
- Updated [roles and permissions table](/en/portal/organization/roles-and-permissions) to include Download Transaction History permissions
- Added new user roles documentation in [Introduction to user roles and permissions](/en/portal/organization/roles-and-permissions)

**Token management**
- Added detailed instructions for self service token listing process in [Supported tokens and chains](/en/portal/supported-tokens)
- Added links to self service token listing documentation from [Asset Wallets](/en/portal/custodial-wallets/asset-wallets/manage) and [MPC Wallets](/en/portal/mpc-wallets/ocw/manage-wallets) management sections

**Glossary updates**
- Added new terms and standardized terminology in [Glossary](/en/portal/glossary)
- Reorganized all entries in [Glossary](/en/portal/glossary) alphabetically
</Update>

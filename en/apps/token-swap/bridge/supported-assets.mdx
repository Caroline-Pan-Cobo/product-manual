## Overview

[Token Swap](/en/apps/token-swap/introduction)'s Bridge feature enables secure cross-chain transfers between the following tokens and chains:

## BTC bridging
BTC can be bridged between the following chains:
- Bitcoin
- Ethereum

## USDC, USDT, and ETH bridging
USDC, USDT, and ETH can be bridged between the following chains:
- Ethereum
- Solana
- Tron
- Op Mainnet
- Arbitrum
- Base
- Polygon
- BNB Chain
- Avalanche

The swap feature currently supports a curated set of tokens for same-chain conversions. This list will be updated as support expands.

<Info>Currently cross-chain swaps are not supported. To move assets across chains, please use the [Bridge](/en/apps/token-swap/bridge/introduction) feature.</Info>

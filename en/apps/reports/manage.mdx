Only admins and operators have the permissions to generate and download reports.

To manage Reports, click <img src="/en/images/portal-apps/reports/apps.svg" className="icon" alt="the Apps icon"/> on Cobo Portal, find Reports, then click **Open** to open Reports.

### Transaction Reports

Under the Transactions tab, you can generate new transaction reports and/or view existing ones.

- **Generate** <img src="/en/images/portal-apps/reports/generate.svg" className="icon" alt="the Apps icon"/>: Create a new transaction report by specifying criteria such as time zone, timeframe, and wallet types.
- **Reset** <img src="/en/images/portal-apps/reports/reset.svg" className="icon" alt="the Apps icon"/>: Clear all current selections and revert to default settings.
- **Search** <img src="/en/images/portal-apps/reports/search.svg" className="icon" alt="the Apps icon"/>: Find a transaction report using its report ID.
- **Download** <img src="/en/images/portal-apps/reports/download.svg" className="icon" alt="the Apps icon"/>: Save a transaction report for offline use.

<img src="/en/images/portal-apps/reports/1.png" className="full_screen" alt="Image showing Reports as a Cobo Portal App."/>

### Asset Holdings Reports

Under the Asset Holdings tab, you can generate new asset holdings reports and/or view existing ones.

- **Generate** <img src="/en/images/portal-apps/reports/generate.svg" className="icon" alt="the Apps icon"/>: Create a new asset holdings report by specifying criteria such as time zone, timestamp, wallet types, and display options (e.g., by assets, by wallets).
- **Reset** <img src="/en/images/portal-apps/reports/reset.svg" className="icon" alt="the Apps icon"/>: Clear all current selections and revert to default settings.
- **Search** <img src="/en/images/portal-apps/reports/search.svg" className="icon" alt="the Apps icon"/>: Find an asset holdings report using its report ID.
- **Download** <img src="/en/images/portal-apps/reports/download.svg" className="icon" alt="the Apps icon"/>: Save a report for offline use.

<img src="/en/images/portal-apps/reports/2.png" className="full_screen" alt="Image showing Reports as a Cobo Portal App."/>

Staking is a [Cobo Portal App](/en/portal/portal-apps/introduction) developed by Cobo. To install Staking, follow the instructions in [Browse and install apps](/en/portal/portal-apps/browse-and-install).

<Info>Currently, the Staking App is available for [MPC Wallets](/en/portal/mpc-wallets/introduction) in [Organization-Controlled Vaults](/en/portal/mpc-wallets/ocw/create-mpc-vault) and supports staking BTC with the Babylon protocol.</Info>

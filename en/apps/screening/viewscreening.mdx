Click <img src="/en/images/common/icon_apps.svg" className="icon" alt="the Apps icon"/> on [Cobo Portal](https://portal.cobo.com/login), find **Screening**, then click **Open** to open screening. 

In the top right corner of the summary section, you can click <img src="/en/images/portal-apps/screening/screeningtime.png"  className="icon" alt="select screening time"  />to select a screening time range. The page will then display corresponding information for the selected time range. By default, the screening time range is set to the latest 30 days.

<img src="/en/images/portal-apps/screening/screeningresult.png" className="screenshot_full_screen" alt="Screening Results"/>

- **Summary**: Display the number of flagged transactions, the number of clear transactions, and the flag rate. 

  **Flagged**: The total number of transactions involving sanctioned addresses.

  **Clear**: The total number of transactions not involving any sanctioned addresses.

  **Flag Rate**: Calculated as the percentage of total transactions that have been flagged, providing insight into the proportion of potentially problematic transactions.

- Flagged transactions list - Displays the flagged transaction list. You can search by Case ID or Transaction ID.

  **Case ID**: The ID generated by the Screening app.

  **Transaction ID**:  This is the identifier obtained from Cobo Portal.

  **From**: Source wallet address. The wallet address marked in red is sanctioned.

  **To**: Target wallet address.The wallet address marked in red is sanctioned.

  **Direction**: Inbound or Outbound.

<Info>
To access this feature:
1. Log in to [Cobo Portal](https://portal.cobo.com/login)
2. Click <img src="/en/images/common/icon_apps.svg" className="icon" alt="the Apps icon"/> to open Cobo Portal Apps
3. Find and open **Screening**
4. Click **Screening Results**
</Info>

- Complete audit trails and action histories

### Available actions

The following actions are available based on the transaction's current status:

| Action | Effect |
| :---- | :---- |
| Approve | • For deposits: Transaction passes screening, funds become available to user<br/>• For withdrawals: Transaction passes screening, proceeds to next stage |
| Approve with Alert | • For deposits: Transaction passes screening, funds become available to user, with additional risk transaction notification<br/>• For withdrawals: Transaction passes screening, proceeds to next stage, with additional risk transaction notification |
| Reject | • For deposits: Transaction fails screening<br/>&nbsp;&nbsp;• Transactions are flagged and funds will be frozen<br/>&nbsp;&nbsp;&nbsp;&nbsp;• *Note about frozen funds: If you are using the [auto token sweeping feature of MPC Wallets](/en/portal/auto-sweep-overview), frozen funds will not be included in sweep operations*<br/>• For withdrawals: Transaction fails screening, withdrawal is rejected<br/>• Risk notification is always sent |
| Manual Review<br/><i>(Coming soon)</i> | • For deposits: Will enable manual review of transactions<br/>• For withdrawals: Will enable manual review of transactions |
| Return<br/><i>(Coming soon)</i> | • For deposits: Will enable returning funds from "to address" to specified address<br/>• For withdrawals: Not applicable |
| Unfreeze<br/><i>(Coming soon)</i> | • Will enable releasing frozen funds, making them available to user<br/>• For withdrawals: Not applicable |

The Token Bridge dashboard provides a comprehensive overview of your cross-chain transactions and activities.

## Overview

### Transaction statistics
- **Total Bridge Value**: Total fiat value of completed transactions (excludes pending or failed transactions).
- **Number of Transactions**: Total count of completed transactions (excludes pending or failed transactions).

## Activity

The Activity section displays your bridge transaction history with the following features:

### Bridge Now
Located to the right of the Filter and Sort button, the "Bridge Now" button opens the Token Bridge dialog where you can:
- Initiate new bridge transactions.
- Select source and destination assets.
- Enter transaction amounts.
- View fees and estimated receipts.

For detailed information about the Token Bridge dialog, see [Bridge Token](/en/apps/bridge/how-to-use/manage/bridge-form).

### Filter and Search
- Search by token name or wallet name
- Filter by transaction status
- Filter by update time
- Sort options:
  - Updated Time: New to Old (default)
  - Updated Time: Old to New

### Transaction list
Each transaction entry shows:
- **Amount**: Token logo, amount, and symbol with fiat value.
- **From/To**: Source chain logo â†’ destination chain logo (hover to view chain names).
- **Wallet**: Custodial Wallet name.
- **Initiator**: Email of the transaction initiator.
- **Status**:
  - Success: Transaction completed.
  - Processing: Transaction in progress.
  - Failed: Transaction failed (due to Cobo Guard rejection, risk controls, or other issues).
- **Updated Time**: Latest transaction update timestamp.
- **Actions**: View button to open detailed transaction information.

### Actions
From the dashboard you can:
- Initiate new bridge transactions (Spender role required).
- View detailed transaction information.
- Export transaction history.
- Monitor transaction status.

### Token Bridge details
Click any transaction to view:
- Detailed status updates.
- Complete transaction history.
- Time stamps for each step.
- Error messages (if any).

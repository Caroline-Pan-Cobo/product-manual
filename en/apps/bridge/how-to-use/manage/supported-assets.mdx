This document details the supported assets and networks for Token Bridge cross-chain operations.

## USDT Bridge

### Networks
- Tron (TRX)
- BNB Chain (BNB)
- Ethereum (ETH)

### Processing Method
- Processed through Binance API
- Supports deposits and withdrawals within Binance
- Automated cross-chain transfers

### Transaction Limits
- Daily Limit per Organization: 10,000 - 5,000,000 USDT
- Platform Daily Limit: 50,000,000 USDT

### Fee Structure
- Fee Rate: 0.1%
- Payment Method: Postpay (charged in billing period)
- No minimum transaction amount

### Network Details
1. **Tron (TRX)**
   - Contract: TRC20
   - Network Status: Active
   - Processing Time: ~10 minutes

2. **BNB Chain (BNB)**
   - Contract: BEP20
   - Network Status: Active
   - Processing Time: ~15 minutes

3. **Ethereum (ETH)**
   - Contract: ERC20
   - Network Status: Active
   - Processing Time: ~20 minutes

## BTC Bridge

### Networks
- Bitcoin Mainnet (BTC)
- Ethereum (ETH/WBTC)

### Processing Method
- Processed through BitGo API
- Supports Mint/Burn operations within BitGo
- Automated conversion between BTC and WBTC

### Transaction Requirements
- Minimum Transaction: 10 BTC
- No maximum limit (subject to available balance)

### Fee Structure (Prepay)

#### Mint (BTC → WBTC)
| Amount Range (BTC) | Fee Rate |
|-------------------|----------|
| 0-100             | 0.35%    |
| 100-300           | 0.30%    |
| 300-500           | 0.26%    |
| 500-1500          | 0.23%    |
| 1500+             | 0.20%    |

#### Burn (WBTC → BTC)
| Amount Range (WBTC) | Fee Rate |
|-------------------|----------|
| 0-100             | 0.50%    |
| 100-300           | 0.30%    |
| 300-500           | 0.23%    |
| 500-1500          | 0.19%    |
| 1500+             | 0.15%    |

### Network Details
1. **Bitcoin Mainnet**
   - Native BTC
   - Network Status: Active
   - Processing Time: T+1 settlement

2. **Ethereum (WBTC)**
   - ERC20 Wrapped Bitcoin
   - Network Status: Active
   - Processing Time: T+1 settlement

## Important Notes

1. **Network Availability**
   - Networks may be temporarily unavailable for maintenance
   - Check dashboard for current status

2. **Processing Times**
   - Times are estimates
   - May vary based on network conditions
   - BTC bridges settle on T+1 basis

3. **Fee Calculations**
   - USDT: Fixed 0.1% postpay
   - BTC: Tiered structure based on amount
   - Fees deducted automatically

4. **Transaction Limits**
   - Subject to change based on market conditions
   - Platform limits apply across all users
   - Individual organization limits may vary

## Set up Google Authenticator

1. Log in to [Cobo Accounts](https://accounts.cobo.com/) and click **MFA Management** in the main menu on the left-hand side.
2. Click **Set Up** next to **Google Authenticator**.

    <img src="/en/images/cobo-accounts/q-HIkg_-w32EVlc6NBiAB8VpXO9FtOrm3Vll9uVWlxFljdf_2VxWloLqVvW6fMJ7T9i97PcPWNStXZtAahn1xVKil-BbVFr2Sl2HQz3DRjRZA10c6EfpPphRhu1WaCd9r9hWwznXWF7Jp3bQKnmVa4o.svg" className="screenshot_full_screen"/>

3. Download and install Google Authenticator.

4. Use Google Authenticator to scan the QR code displayed on Cobo Accounts. 

    <Note>Please save this QR code, in case you need to recover Google Authenticator in case of device loss or replacement. If you don't save this QR code and need to reset Google Authenticator, please refer to [Reset Google Authenticator](/en/accounts/reset-google-authenticator).</Note>

5. Enter the 6-digit Google Authenticator code into the text box on Cobo Accounts, then click **Submit**.

6. The Google Authenticator status will be updated to **Enabled** on the **MFA Management** dashboard.

## Set up security keys

<Note>A security key is a secure physical device used to assist in verifying your identity during MFA verification. You can add up to 10 security keys to your Cobo Accounts.</Note>

1. Log in to [Cobo Accounts](https://accounts.cobo.com/) and click **MFA Management** in the main menu on the left-hand side.
2. Click **Add** next to **Security Keys**.

    <img src="/en/images/cobo-accounts/q-HIkg_-w32EVlc6NBiAB8VpXO9FtOrm3Vll9uVWlxFljdf_2VxWloLqVvW6fMJ7T9i97PcPWNStXZtAahn1xVKil-BbVFr2Sl2HQz3DRjRZA10c6EfpPphRhu1WaCd9r9hWwznXWF7Jp3bQKnmVa4o.svg" className="screenshot_full_screen"/>

3. On the next screen, click **Add** next to **Security keys**.
4. To successfully add a security key, you must insert the device and touch it.

    <img src="/en/images/cobo-accounts/g3FHKpCEbBANaxRHdKCyd-4XWe5pOpbIAyGZ5Qlz16FeW0luzTiOkR9S8gZcBXSC7-4oIlyvAJaiXD31ZblZM9y1Fuf1zIZC5T-1Q884aNfYSjLh7s-QFH05_kTtn5tXJfI99TzzMbbgEL7Q3GGed1M.png" className="screenshot_modal"/>

5. You can review the details of your newly added security key, including its creation time and the last time it was used.

    <img src="/en/images/cobo-accounts/y8XYrmZVMGjIxCn2rMr0U-Yy8ZLu0U2zt760lRR7lVdApX0VpDk0562DTlpWhicDSjr4jwaoaUCS88H0sSUmDBGQ2FEWX5VIIQK9SKWnmkrZzjJft8OT-7XZFLTGamtLn7Zxoj7D1M2L_Fle5CoHWck.png" className="screenshot_modal"/>

title: "Assign user roles to members"
sidebarTitle: "Assign user roles to members"
---

You can assign user roles either to new or existing members.

### **Assign user roles to new members**

While inviting new members, you can assign user roles to them under Role Settings.

### **Assign user roles to existing members**

You can assign user roles to existing members by clicking **Organization** > **Settings** > **Members** > **Action (...)** > **Edit User Role**.

Follow the steps below to create an MPC wallet.

1. Log into Cobo Portal.
2. Click the Wallet icon, then click **MPC**.
3. Select the vault that you created in [Create an MPC vault](/cobo-portal/wallets/mpc-wallet/organization-controlled-wallet/set-up/create-mpc-vault).
4. Click **Create Wallet**.
5. Name your wallet, then click **Create**.

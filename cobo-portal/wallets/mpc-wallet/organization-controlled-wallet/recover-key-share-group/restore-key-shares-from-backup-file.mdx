**Note**: If the locally stored private key share is lost, the key share holder can restore it using the backed-up data file, as long as the file remains intact and accessible.

<u>**For Mobile Co-signers**</u>

For more information on how to restore your private key shares, please refer to [Restore your key shares from iCloud](/cobo-guard/restore-your-key-shares-from-icloud).

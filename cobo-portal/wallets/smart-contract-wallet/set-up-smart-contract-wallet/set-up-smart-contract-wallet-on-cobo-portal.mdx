Now that you are a Cobo Argus user who has already set up Safe&#123;Wallet&#125; with Cobo Safe enabled. Follow the instructions below to set up a Smart Contract Wallet on Cobo Portal.

To set up a Smart Contract Wallet on Cobo Portal, click [Insert icon]  > **Smart Contracts** > **Import**.

**(Add screenshot here)**

Select Safe&#123;Wallet&#125;, then click **Next**.

**(Add screenshot here)**

Cobo Portal will start to import your Safe&#123;Wallet&#125; from Cobo Argus.

**(Add screenshot here)**

Select one or more Safe&#123;Wallet&#125; you want to import, then click **Import**.

**(Add screenshot here)**

Click **OK** when you see the Congratulations! dialog.

**(Add screenshot here)**

On Cobo Portal, confirm that your Smart Contract Wallet has been set up.

title: "生成 Bybit API keys"
sidebarTitle: "Bybit - API Keys"
---

按以下步骤为您的 Bybit 账户生成 **API Key** 和 **API Secret**：

1. 登录到您的 Bybit 账户。
2. 导航至**我的资料** > **账户和安全** > **API**。
3. 选择**新建密钥**。
4. 选择**系统生成的 API 密钥**。

<img src="/cn/images/exchange-wallet/bybit.png" className="screenshot_modal"/>

5. 选择 **API 交易**。
6. 填写 API Key 的名称 = **Cobo**。
7. 选择以下 **2 项 API Key 权限**：
    - 读写
    - 仅允许已授权 IP 访问 OpenAPI
8. 在 API Key 权限下填写 Cobo 的 IP 地址 = **18.177.176.209**。
9. 勾选**读写**下的 4 个选项：
    - 账户转账
    - 子账户转账
    - 提币
    - 交易所历史记录
10. 点击**提交**。

您已成功生成 **API Key** 和 **API Secret**，现在可以返回到 Cobo Portal 界面来完成后续设置。有关更多信息，请参阅该[指南](https://www.bybit.com/en/help-center/article/How-to-create-your-API-key)。

title: "生成 Deribit API keys"
sidebarTitle: "Deribit - API Keys"
---

按以下步骤为您的 Deribit 账户生成 **Client ID** 和 **Client Secret**：

1. 登录到您的 Deribit 账户。
2. 导航至**我的账户** > **API**。
3. 选择 **+ 添加新密钥**。

<img src="/cn/images/exchange-wallet/deribitcn.png" className="screenshot_full_screen"/>

4. 选择账户 = **读取**。
5. 选择钱包 = **读写**。
6. 在 IP 白名单下填写 Cobo 的 IP 地址 = **18.177.176.209**。
7. 点击**创建新密钥**。
8. 创建 API Key 后，点击**加载 Key**。

您已成功生成 **Client ID** 和 **Client Secret**，现在可以返回到 Cobo Portal 界面来完成后续设置。

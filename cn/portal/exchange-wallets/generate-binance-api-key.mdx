sidebarTitle: "Binance - API Keys"

按以下步骤为您的币安账户生成 **API Key** 和 **API Secret Key**：

1. 登录您的 Binance 账户。
2. 导航至**账户** > **API 管理**。
3. 点击**创建 API**。
4. 选择**系统生成**，然后点击**下一步**。

<img src="/cn/images/exchange-wallet/binance1.png" className="screenshot_modal"/>

5. 勾选以下**4 条 API 限制**：
    - 启用读取
    - 启用内部转账
    - 允许通用转账
    - 启用提币
6. 选择**限制只对受信任 IP 的访问（推荐）**。
7. 在受信任 IP 下填写 Cobo 的 IP 地址： **18.177.176.209**。

<img src="/cn/images/exchange-wallet/binance4.png" className="screenshot_full_screen"/>

8. 点击**保存**。

您已成功生成 **API Key** 和 **API Secret Key**，现在可以返回到 Cobo Portal 界面来完成后续设置。有关更多信息，请参阅该[指南](https://www.binance.com/en/support/faq/how-to-create-api-keys-on-binance-360002502072)。

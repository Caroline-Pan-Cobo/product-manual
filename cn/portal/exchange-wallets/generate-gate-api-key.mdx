按以下步骤为您的 Gate 账户生成 **API Key** 和 **API Secret Key**：

1. 登录到您的 Gate 账户。
2. 导航至**我的资料** > **API 管理**。
3. 选择**创建 API Key**。
4. 填写 API Key 备注 = **Cobo**。
5. 在 IP 地址权限下填写 Cobo 的 IP 地址 = **18.177.176.209**。
6. 点击**提交**。
7. 选择 **API v4 Key**。
8. 选择**交易账户**。
9. 开启以下 **5 项权限**：
    - 现货：只读
    - 钱包：读写
    - 提币：读写
    - 子账户：只读
    - 账户：只读

<img src="/cn/images/exchange-wallet/gate.png" className="screenshot_modal"/>
10. 点击**确认**。

您已成功生成 **API Key** 和 **API Secret Key**，现在可以返回到 Cobo Portal 界面来完成设置。有关更多信息，请参阅该[指南](https://www.gate.io/help/guide/faq/17521/how-to-utilize-api)。

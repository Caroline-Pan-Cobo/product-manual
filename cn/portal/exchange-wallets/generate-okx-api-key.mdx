title: "生成 OKX API key”"
sidebarTitle: "OKX - API Keys"
---

按以下步骤为您的 OKX 账户生成 **API Key** 和 **API Secret**：

1. 登录到您的 OKX 账户。
2. 导航至**交易所** > **我的账户** > **API**。
3. 选择 **++ 创建 API Key**。
4. 填写 API 名称 = **Cobo**。
5. 填写口令 = **输入您所选的安全口令**。
6. 在绑定 IP 地址下填写 Cobo 的 IP 地址 = **18.177.176.209**。
7. 选择 **API 交易**。
8. 勾选以下 **3 项权限**：
    - 读取
    - 提现
    - 交易
<img src="/cn/images/exchange-wallet/okx.jpg" className="screenshot_modal"/>

9) 点击**确认**。

您已成功生成 **API Key** 和 **API Secret**，现在可以返回到 Cobo Portal 界面来完成后续设置。

<Note>前提条件：请根据[快速入门指南](/cn/portal/quick-start-guide)设置您的账户并完成其他所需步骤。</Note>

通过以下简单步连接您的交易所账户，在 Cobo Portal 上设置您的交易所钱包：

1) 登录 Cobo Portal。
2) 点击 <img src="/cn/images/exchange-wallet/ew3.svg" className="icon"></img>，然后选择**交易所钱包**。
3) 点击 **+ 交易所账户**，或者点击特定交易所下的**连接账户**。
4) 前往交易所平台，在 API Key 设置中将 Cobo IP (**18.177.176.209**) 列入白名单，并生成 API Key 和 API Secret。有关各个交易所的详细说明，请参阅后续页面。
5) 填写必填信息（名称、API Key、API Secret 和口令），然后点击**下一步**。

<img src="/cn/images/exchange-wallet/ew4.png" className="screenshot_modal"/>

6) （可选）如果适用，请选择要需要连接的交易所子账户。
7) 点击**连接**完成设置。设置成功后，您将在交易所钱包的详情页面看到已经连接的交易所账户。

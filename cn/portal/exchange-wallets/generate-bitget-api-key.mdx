title: "生成 Bitget API keys"
sidebarTitle: "Bitget - API Keys"
---

按以下步骤为您的 Bitget 账户生成 **API Key** 和 **API Secret Key**：


1. 登录到您的 Bitget 账户。
2. 导航至**个人中心** > **API 管理**。

<img src="/cn/images/exchange-wallet/bitget.png" className="screenshot_full_screen"/>

3. 点击**新建 API**。
4. 填写所有必填字段。注意，将交易所连接到 Cobo Portal 时需要您输入口令。
5. 选择权限 = **读写**。
6. 选择现货 = **交易**。
7. 选择钱包 = **提币**。
8. 在 IP 白名单下填写 Cobo 的 IP 地址 = **18.177.176.209**。
9. 点击 **下一步**。

您已成功生成 **API Key** 和 **API Secret Key**，现在可以返回到 Cobo Portal 界面来完成设置。有关更多信息，请参阅该[指南](https://www.bitget.com/academy/bitget-api-trading-guideline)。

---
title: "生成 Bitget API key"
sidebarTitle: "Bitget"
---

当您将 Bitget 账户关联到交易所钱包时，需要提供 API key、API secret key 和口令。此外，还需要将 Cobo 加入 IP 白名单，并为 API key 分配特定权限。

本文介绍如何在 Bitget 上生成 API key，包括将 Cobo 加入 IP 白名单和设置权限。

<Note>内容如有变更，以 Bitget 网站为准。有关更多详细信息，请参阅 <a href="https://www.bitget.com/zh-CN/academy/bitget-api-trading-guideline">Bitget API 创建指南</a>。</Note>

1. 登录到您的 Bitget 账户。
2. 导航至**个人中心** > **API 管理**。
    <img src="/en/images/exchange-wallet/bitget api.png" className="screenshot_modal" alt="The API Management page" />
3. 点击**新建 API** 打开对话框。
4. 在**新建 API key** 对话框中，填写所有必填项。 
    - 按要求输入口令。
    - 在 **IP 白名单**字段中，输入 Cobo 的 IP 地址 `18.177.234.7` 及 `18.180.16.190`。
    - 选择以下权限：
        - **期货** > **持仓** 
        - **现货** > **交易** 
        - **利润** > **交易** 
        - **钱包** > **转账**和**提币**
    <img src="/en/images/exchange-wallet/bitget_per_1.png" className="screenshot_modal" alt="The required permissions list" />
    <img src="/en/images/exchange-wallet/bitget_per_2.png" className="screenshot_modal" alt="The required permissions list" />
5. 点击**下一步**生成 API key。

API key 现已生成。您可以前往 Cobo Portal，使用 API key、API secret key 和口令关联您的 Bitget 账户。如欲了解如何关联账户，请参见[设置交易所钱包](/cn/portal/exchange-wallets/set-up)。

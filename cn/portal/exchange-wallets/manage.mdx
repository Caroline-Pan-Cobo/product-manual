交易所钱包业务数据概览提供了对每个交易所整体余额的全面概览。要了解详细信息，可在 Cobo Portal 上点击相应的交易所，按账户类型查看余额细分情况。此外，通过点击相应的交易所账户，还可以查看每个账户中资产类型的细分情况。

点击相应的图标，即可轻松管理您交易所钱包中已经连接的账户：

- **充币** <img src="/cn/images/exchange-wallet/10.png" className="icon"></img>：向交易所账户充币。充币只能在主账户上进行。
- **提币** <img src="/cn/images/exchange-wallet/11.png" className="icon"></img>：从交易所账户提币。仅特定交易所账户类型允许将资金从主账户提取到外部地址。有关更多信息，请参阅[从交易所账户转账](/cn/portal/transfers/from-exchange-wallets)章节。
- **转入** <img src="/cn/images/exchange-wallet/12.png" className="icon"></img>：在主账户和子账户之间转移资产，或在不同账户类型之间（如资金账户和交易账户之间）转移资产。
- **查看交易记录** <img src="/cn/images/exchange-wallet/14.png" className="icon"></img>：查看交易所钱包内的动态，包括交易历史和待处理交易的详细历史记录。
- **API 管理** <img src="/cn/images/exchange-wallet/15.png" className="icon"></img>：前往到对应的交易所，生成 API Key 和 API Secret Key。
- **刷新** <img src="/cn/images/exchange-wallet/16.png" className="icon"></img>：更新并同步您的资产余额，以确保与关联账户的最新数据保持一致。
- **解除关联** <img src="/cn/images/exchange-wallet/17.png" className="icon"></img>：取消与已连接账户的关联。
- **关联子账户** <img src="/cn/images/exchange-wallet/18.png" className="icon"></img>：将子账户关联到对应的主账户。
- **过滤** <img src="/cn/images/exchange-wallet/19.png" className="icon"></img>：根据链、资产类型或余额筛选钱包，定制视图。
- **排序** <img src="/cn/images/exchange-wallet/20.png" className="icon"></img>：按最后活跃时间或账户余额进行账户排序，整理账户。
- **搜索** <img src="/cn/images/exchange-wallet/21.png" className="icon"></img>：通过钱包名称、地址或资产类型惊醒搜索，快速找到需要查看的交易所钱包或代币。

---
title: "生成 Bitmart API key"
sidebarTitle: "Bitmart"
---

当您将Bitmart 账户关联到交易所钱包时，需要提供 API key、Secret key 和备注。此外，还需要将 Cobo 加入 IP 白名单，并为 API key 分配特定权限。

本文介绍如何在 Bitmart 上生成 API key，包括将 Cobo 加入 IP 白名单和设置权限。

<Note>内容如有变更，以 Bitmart 网站为准。详情请参阅<a href="https://support.bitmart.com/hc/zh-cn/articles/360016076854-%E5%A6%82%E4%BD%95%E5%88%9B%E5%BB%BAAPI">如何创建 API</a>。</Note>

1. 登录到您的 Bitmart 账户。
2. 在顶部导航栏上，点击你的资料，然后点击**账户**以显示**账户**选项卡。
3. 在**安全设置**中，点击**API**行中的**设置**，显示**新建 API** 页面。
    <img src="/en/images/exchange-wallet/bitmart.png" className="screenshot_modal" alt="The Create New API page" />
4. 在**备注**部分，在字段中输入备注。
5. 在**权限**部分，选择**提币**。
6. 在**绑定 IP 地址**部分，输入 Cobo 的 IP 地址 `18.177.234.7` 及 `18.180.16.190`。
7. 点击**提交**生成 API key。

成功生成 API key 后，您将看到一个 API key 和一个 Secret key。请小心保存 Secret key，因为它仅以明文显示一次。

API key 现已生成，且设置已完成。您可以前往 Cobo Portal，使用 API key、Secret key 和备注关联您的 Bitmart 账户。如欲了解如何链接账户，请参见[设置交易所钱包](/cn/portal/exchange-wallets/set-up)。

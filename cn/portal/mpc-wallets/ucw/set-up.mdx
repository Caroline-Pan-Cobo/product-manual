## 概览
终端用户钱包主要通过 [Cobo Wallet-as-a-Service (WaaS) 2.0 API](https://www.cobo.com/developers/v2/guides/overview/introduction) 和 [UCW SDK](https://www.cobo.com/developers/v2/developer-tools/quickstart-ucw-ios) 进行设置。我们还提供了示例代码帮助您快速开始。有关实现自有终端用户钱包的完整指南，请参阅 Developer Hub 中的 [Get started with MPC Wallets (User-Controlled Wallets)](https://www.cobo.com/developers/v2/guides/mpc-wallets/get-started-ucw)。

设置终端用户钱包后，您可以按照后续章节中的说明，使用 Cobo Portal 对其进行管理。

创建一个或多个 [3 种私钥分片组](/en/portal/mpc-wallets/ocw/create-key-share-groups)后，最好让私钥分片持有方备份自己的私钥。私钥分片持有方可以采用不同的方式备份私钥分片，具体取决于私钥分片是使用移动端签名人还是 API 签名人生成。

## 移动端签名人
私钥分片持有方可以使用他们用来确认私钥组创建过程的 Cobo Guard App 来备份在其 Cobo Guard 上创建的任何私钥分片。请参见[生成和备份私钥分片](/en/guard/generate-and-back-up-key-shares)。

## API 签名人
私钥分片持有方可以使用 API 签名人来备份其私钥分片。后续我们将在开发者中心提供相关说明。

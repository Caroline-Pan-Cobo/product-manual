TSS Node 是 Cobo Portal 为客户提供的程序，用于管理和使用私钥分片。它有两个主要功能：1) 生成和管理私钥分片；2) 完成交易签署。TSS Node 需要以下网络环境以实现最佳运行：

- 无需入站连接
- 服务器正在安装或升级时：
  - 能够访问系统的 apt 源，以安装所有必要的依赖项（仅适用于使用 SGX 服务器或通用服务器的情况）
  - 能够访问 Docker Engine 安装文件，以安装 Docker Engine
  - 能够访问 docker.io 注册表（公共 Docker 注册表）
  - 能够访问 Intel 网站以下载驱动程序（适用于 SGX 服务器）
- 服务器运行时：
  - 开发环境：能够访问 ws.tss.dev.cobo.com 的 443 端口
  - 正式环境：能够访问 ws.tss.cobo.com 的 443 端口
  - 能够访问客户端设置的 callback 服务器

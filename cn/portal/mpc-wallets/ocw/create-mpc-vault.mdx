<Note>注意：只有拥有[操作员](/cn/portal/organization/roles-and-permissions)角色的用户能够创建 Vault。</Note>

Vault 是一个钱包集合，这些钱包具有相同的根扩展私钥。系统将自动为您创建一个默认 Vault。若您希望创建其他 Vault，请按照以下步骤操作：

1. 登录 [Cobo Portal](https://portal.cobo.com/login)。

2. 单击默认 Vault 旁边的 <img src="/cn/images/mpc-wallet/switch.svg" className="icon"></img> 图标。

<img src="/cn/images/mpc-wallet/switch.png" className="screenshot_full_screen"/>

3. 在弹出的对话框中，点击**返回 Vault 列表**。

4. 点击页面右上角的**创建 Vault**。

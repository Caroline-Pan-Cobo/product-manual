- **免费试用**：获得 14 天免费试用后，您可以免费体验 Cobo Portal 并访问相应的钱包。一旦免费试用到期或达到免费试用的用量限制，则必须购买付费套餐。

- **专业版套餐**：专业版套餐专门面向使用需求较高和需要定制化的企业。如果对我们的专业版套餐感兴趣，[请联系我们的销售团队](https://www.cobo.com/book-demo/?utm_source=product-manual&utm_medium=product&utm_campaign=pricing-plan-intro&utm_content=enterprise-contact-sales)，他们将协助您找到最符合您需求的定制套餐。
- **免费版套餐**: 在使用免费版套餐之前，您需要先购买全托管钱包或 MPC 钱包的付费套餐。如果您所有的付费套餐都因为欠费而被暂停服务，您的免费版套餐也会被暂停服务。

| WaaS | 使用 Cobo 的 WaaS API 以编程方式创建数字资产钱包的企业 | 交易所、加密原生公司、OTC 和券商以及零售平台 | 提币金额和钱包地址数量的额度较高
| 财资管理 | 希望直接在 Cobo Portal 上保护和管理其数字资产的企业 | DeFi 和质押基金、交易公司、机构投资方和金融服务 | 资产托管金额的额度较高

## 使用项目

下表概述了付费套餐中包含的使用项目。请注意，具体的使用项目可能因您选择的套餐而异。

- **累计提币金额**：您提取到外部地址的资产总量。全托管钱包、MPC 钱包和智能合约钱包内部和之间的转账不计入累计提币金额。
- **资产托管金额**：您存储在 Cobo Portal 中的资产总量。
- **钱包地址数**：您生成的地址数量。
- **可用公链数**：您已启用的链的数量。链一旦启用，即视为正在使用，并将计入套餐使用量。
- **API 调用次数**：您发送的 WaaS API 请求数量。该数量在整个团队范围内计算。
- **用户成员数**：您所在团队的用户数量。该数量在整个团队范围内计算。

对于 14 天免费试用，一旦试用期结束或达到用量限制，您将只能浏览您的 MPC 钱包和交易所钱包。您必须购买付费套餐才可重新使用完整功能。

对于付费套餐，一旦达到用量限制，您需要按照相应的单价为超出部分支付额外使用费用。此时，您可以选择以折扣价购买用量包或升级至更高级别的套餐。

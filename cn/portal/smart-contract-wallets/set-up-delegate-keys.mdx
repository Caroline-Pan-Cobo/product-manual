sidebarTitle: "设置 Delegate Key"
---

Delegate Key 用于发起（但不签署）代表 Safe&#123;Wallet&#125; 签名人的多签交易。Delegate Key 是一个由 Cobo 分配并在 Cobo Portal 上创建的专用地址，它是设置链上交易风控规则的必要条件。请按照以下说明设置您的 Delegate Key。

1. 登录 [Cobo Portal](https://portal.cobo.com/login)。
2. 点击 <img src="/cn/images/smart-contract-wallet/button_wallet.svg" className="icon"></img> > **智能合约钱包** > 您要使用的 Safe&#123;Wallet&#125; 旁边的 <img src="/cn/images/smart-contract-wallet/icon_key.svg" className="icon"></img>。
3. 点击 <img src="/cn/images/smart-contract-wallet/button_copy.svg" className="icon"></img> 以复制弹出窗口中显示的超链接。然后，将超链接发送给您的任意一位 Safe&#123;Wallet&#125; 签名人以进行签署。或者，点击**复制全部并关闭**以复制所有信息，包括 Safe&#123;Wallet&#125; 地址、Delegate Key 和 URL。
<img src="/cn/images/smart-contract-wallet/3.png" className="screenshot_modal"></img>
4. （如果您是 Safe&#123;Wallet&#125; 签名人）
- 访问提供的链接，并连接与 Safe&#123;Wallet&#125; 关联的钱包。
- 在弹出窗口中，检查自动填写的字段，并为 Delegate Key 输入一个标签。然后，点击**签署消息**。
- 在钱包内完成签名。
<img src="/cn/images/smart-contract-wallet/4.png" className="screenshot_modal"></img>
5. 新的 Delegate Key 将显示在主页上。

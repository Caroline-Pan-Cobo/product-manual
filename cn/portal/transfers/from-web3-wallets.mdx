## 前提条件

- 推荐您在从 Web3 钱包转账前设置[链下交易风控规则](/cn/portal/risk-controls/off-chain-tx-set-transfer)。
- 您已向 Web3 钱包[充币](/cn/portal/custodial-wallets/transfers)。

## 步骤

1. 登录 [Cobo Portal](https://portal.cobo.com/login)。

2. 要打开 **转账** 对话框，请点击 Cobo Portal 主菜单中的 <img src="/en/images/transfers/transfer-button.svg" className="icon" alt="转账按钮" />。或者，您也可以在 Web3 钱包页面点击 <img src="/en/images/custodial-wallet/withdraw-icon.png" className="icon"/>，相应的代币或地址将自动为您预选。

    <img src="/cn/images/transfers/transfer-button-location.png" className="screenshot_modal" alt="转账图标位置截图" />
    
3. 选择转账的代币类型。要启用新的代币类型，请到您的 Web3 钱包页面，点击**添加代币**。
4. 在**出金地址**下，对于**钱包类型**，选择 **Web3 钱包**。对于钱包地址，可以从下拉菜单中选择，或通过钱包名称、地址或标签进行搜索。结果中将列出通过 Cobo Portal 和 WaaS 2.0 API 创建的Web3 钱包。
5. 从地址簿或您在 Cobo Portal 上的钱包中选择接收地址，也可以手动输入地址。
        - **地址簿**<img src="/en/images/common/icon_address_book.svg" className="icon" alt="Address Book icon" />：仅列出与所选代币的区块链网络相关联的地址。
        - **我的钱包**：选择钱包类别，然后点击您希望转入的钱包。
    
    如果该转账可以作为 [Cobo Loop](/cn/portal/custodial-wallets/cobo-loop) 转账执行，UI 上会出现相关信息栏，如以下截图所示。
        
        <img src="/cn/images/custodial-wallet/web3/web3-wallet-cobo-loop.png" className="screenshot_full_screen" alt="Cobo Loop 信息栏截图" />

    <Info>如果您使用 WaaS 2.0 API，可以使用 [Check Cobo Loop Transfers](https://www.cobo.com/developers/v2/api-references/transactions/check-loop-transfers) 操作来确定交易是否可以作为 Cobo Loop 转账执行。或者直接调用 [Transfer token](https://www.cobo.com/developers/v2/api-references/transactions/transfer-token) 操作，强制将交易作为 Cobo Loop 转账执行。</Info>

6. 输入转账金额。
7. 查看交易费用。交易费用包括网络费用。该费用是固定的，无法修改。相关费用将从选择的“转出”地址扣除，具体金额因转账的代币而异。
8. <MySnippet />
9. 点击**提交**。
10. 在 Cobo Guard 中批准该请求。如果该地址为新地址，不在您的地址簿或 Cobo Portal 上创建的钱包中，系统将提示您将该地址添加到地址簿。

## 后续步骤

从Web3 钱包进行转账后，您可以在**交易记录**中查看其交易状态。闪电图标 <img src="/en/images/custodial-wallet/icon_loop.png" className="icon" alt="闪电图标" /> 表示该交易是作为 Cobo Loop 转账执行的，如以下截图所示。
<img src="/cn/images/transfers/loop_history.png" className="screenshot_full_screen" alt="交易记录截图" />

如果您使用 WaaS 2.0 API，可以调用 [Get transaction information](https://www.cobo.com/developers/v2/api-references/transactions/get-transaction-information) 或 [List all transactions operations](https://www.cobo.com/developers/v2/api-references/transactions/list-all-transactions) 并检查响应中的`is_loop`属性。
此外，您还可以使用 [Cobo Loop Explorer](https://loop.top/) 通过 Loop ID（即该交易的交易哈希）来追踪 Cobo Loop 转账的状态。

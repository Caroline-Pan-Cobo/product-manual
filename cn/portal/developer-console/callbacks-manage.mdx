只需点击相应的按钮或图标，即可在开发者控制台中管理 callback endpoint：

- **注册 callback endpoint**：注册 callback endpoint，用于在 Cobo Portal 上接收警报。
- **编辑** <img src="/cn/images/developer-console/icon_edit.svg" className="icon"></img>：修改所选 callback URL 的字段。
- **撤销** <img src="/cn/images/developer-console/icon_revoke.svg" className="icon"></img>：永久禁用通过所选 callback URL 发送的所有 Cobo Portal 系统通知。
- **筛选** <img src="/cn/images/developer-console/icon_filter.svg" className="icon"></img>：根据事件的状态和类型，按照 callback URL 定制视图。
- **搜索** <img src="/cn/images/developer-console/icon_search.svg" className="icon"></img>：通过搜索 URL 或事件描述，快速找到 callback URL。

Get a quick overview of your total API requests, webhook events, callback messages, and errors made over the past month.

- **API Requests**: Track the total API requests across various time periods (daily, weekly, monthly).
- **API Request Distribution**: Track the distribution of your API requests or errors according to different method types (Delete, Put, Post, and Get) across various time periods (daily, weekly, monthly).
- **Top 5 API Request**: Track the top 5 URL sources with the most number of API requests across various time periods (daily, weekly, monthly).
- **Webhook Events**: Track the total webhook events across various time periods (daily, weekly, monthly).
- **Callback Messages**: Track the total callback messages across various time periods (daily, weekly, monthly).
- **Recent Errors**: Track the recent API errors made. Click More to view the detailed log of your errors.

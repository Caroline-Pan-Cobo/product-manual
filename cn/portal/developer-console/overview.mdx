快速概览过去一个月内的 API 请求、Webhook 事件、callback 消息和错误总数。

- **API 请求**：跟踪不同时间段（每天、每周、每月）的 API 请求总数。
- **API 请求分布**：根据不同的请求方式（Delete、Put、Post、Get），跟踪不同时间段（每天、每周、每月）内 API 请求或错误的分布情况。
- **排名前 5 的 API 请求**：跟踪不同时间段（每天、每周、每月）内 API 请求数量排名前 5 的 URL 源。
- **Webhook 事件**：跟踪不同时间段（每天、每周、每月）内 Webhook 事件总数。
- **Callback 消息**：跟踪不同时间段（每天、每周、每月）的 callback 消息总数。
- **最近的错误**：跟踪最近发生的 API 错误。单击“更多”可查看错误的详细日志。

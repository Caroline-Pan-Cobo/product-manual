## 前提条件
您已按照 [Set up a callback or webhook endpoint](https://www.cobo.com/developers/v2_cn/guides/webhooks-callbacks/set-up-endpoint) 中的说明设置了 Webhook Endpoint。

## 步骤

1. 登录 [Cobo Portal](https://portal.cobo.com/login)。
2. 在主菜单中点击开发者图标 <img src="/cn/images/common/icon_developer.svg" className="icon" alt="开发者图标" /> 并点击 **Webhook** 标签。
3. 点击**注册 Webhook Endpoint** ，出现以下对话框。

    <img src="/cn/images/developer-console/register_webhook_endpoint.png" className="screenshot_modal" alt="注册 Webhook Endpoint 对话框" />

4. 输入 URL。
5. 输入 Webhook 的描述。
6. 选择您想要接收的事件通知类型。
    <Note>欲了解更多事件类型的详情，请参阅 [Webhook event types](https://www.cobo.com/developers/v2_cn/guides/webhooks-callbacks/webhook-event-type#webhook-event-types)。</Note>
7. 点击**注册**。

至此，您已完成 Webhook Endpoint 的注册。Webhook Endpoint 一旦注册后将无法删除。但是如果您不再需要，您可以让它失效，这意味着永久禁用其功能。有关管理 Webhook Endpoint 的更多详细信息，包括如何让它们失效，请参阅[管理 Webhook Endpoint](/cn/portal/developer-console/webhooks-manage)。

按照以下简单几步即可注册 Webhook Endpoint：

1. 登录 [Cobo Portal](https://portal.cobo.com/login)。
2. 在左侧导航菜单中，点击**开发者** > **Webhook**。
3. 点击**注册 Webhook Endpoint**。
4. 输入 URL 链接。
5. 输入 Webhook 的描述（最多 30 个字符）。
6. 选择希望接收的事件通知类型（可选择多个）。
7. 单击**注册**。

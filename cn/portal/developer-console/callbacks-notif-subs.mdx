title: "Available event notification subscriptions"
sidebarTitle: "Available event notification subscriptions"
---

Create a callback event notification with the following user-initiated triggers:

- **Exchange Wallet Withdrawal**: Trigger a response for a withdrawal made from the Exchange Wallet.
- **Custodial Wallet Withdrawal**: Trigger a response for a withdrawal made from the Asset/Web3 Wallet.
- **Smart Contract Wallet Withdrawal**: Trigger a response for a withdrawal made from the Safe Wallet.
- **MPC Wallet Withdrawal**: Trigger a response for a withdrawal made from the Organization-controlled / User-controlled Wallet.

点击相应的图标，即可在开发者控制台中管理 API 日志：

- **搜索** <img src="/cn/images/developer-console/icon_search.svg" className="icon"></img>：通过搜索 API Key、IP 地址、请求 ID 或 API Endpoint，快速找到 API 日志。
- **勾选框**：根据 API 请求的状态选择 API 日志的显示方式。
- **时间** <img src="/cn/images/developer-console/button_time.svg" className="icon"></img>：选择显示某一时间段内的所有 API 请求。
- **方式** <img src="/cn/images/developer-console/button_method.svg" className="icon"></img>：从下拉菜单中选择 API 请求的请求方法。
- **更多** <img src="/cn/images/developer-console/button_more.svg" className="icon"></img>：输入 API Key、API Endpoint、IP 地址、状态码或 API 请求时间，查找所需的日志详情。
- **全部清除** <img src="/cn/images/developer-console/button_clear all.svg" className="icon"></img>：移除所有已选中的条件。

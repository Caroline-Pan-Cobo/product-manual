按照以下步骤注册 Callback Endpoint：

1. 登录 [Cobo Portal](https://portal.cobo.com/login)。
2. 在左侧导航菜单中，点击**开发者** > **Callbacks**。
3. 点击**注册 Callback Endpoint**。
4. 输入 Callback URL。
5. 输入 Callback 描述（最多 30 个字符）。
6. 单击**注册**。

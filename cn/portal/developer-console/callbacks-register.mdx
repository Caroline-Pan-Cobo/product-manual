Register a callback following these simple steps:

1. Log into Cobo Portal.
2. On the left navigation menu, click **Developer** > **Callbacks**.
3. Click **Register a Callback**.
4. Enter the callback URL.
5. Enter a description for the callback (maximum 30 characters).
6. Select the types of event notification (able to select more than one).
7. Click **Confirm**.

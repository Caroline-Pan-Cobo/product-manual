## 概览
通过概览，您可以查看团队一系列实时指标及相关的可视化呈现，包括 API 请求、API 请求分布、排名前 5 的 API 请求、Webhook 和 Callback。这些洞察提供了有关用户使用模式的宝贵信息，便于优化资源并主动发现潜在问题。

## API Key
管理团队的 API Key，这对于在 Cobo Portal 上验证和授权 API 请求非常重要。您可以在此页面创建、编辑、撤销 API Key，监控 API Key 的使用情况。

## Webhook
Webhook 支持实时事件通知。配置和管理 Webhook endpoint，确保指定团队用户在发生关键事件时随时了解相关信息。

## Callback
callback 可响应特定事件，执行相应的代码。配置和管理 callback 信息，确保用户发起的特定事件会触发预定的响应。

## API 日志
一个全面的日志查看器，提供与团队有关的所有 API 通信请求的详细信息。该功能对于调试、故障排除和审计非常重要，可确保您全面了解团队与用户之间的交互情况。

## Webhook 事件
Cobo Portal 与集成应用程序之间发生的所有 Webhook 事件的详细视图。该功能允许您跟踪和监控特定事件，使您能够监督所有事件通知的状态。

## Callback 消息
用于监控所有触发的 callback 消息的状态。该功能可用于调试并确保您的团队妥当处理用户发起的响应。

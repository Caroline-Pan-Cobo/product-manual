Manage the API keys in your Developer Console easily by clicking on the corresponding icons:

- **Create an API key**: Create an API key with predefined user roles, permissions, and IP whitelists, for authorized interactions with Cobo Portal.
- **Edit**: Make changes to the fields of the selected API key.
- **Revoke**: Permanently disable all Cobo Portal communications with the selected external API key.
- **Filter**: Tailor your view by API keys based on the status and type of keys.
- **Search**: Quickly locate your API keys by searching with the key name or related keywords.

---
title: "Why create API keys on Developer Console?"
sidebarTitle: "Why create API keys on Developer Console?"
---

Owners can create API keys on the Developer Console and assign specific user roles, permissions, and IP whitelists. Multiple API keys can be created depending on the different usage scenarios.

- **Specify User Roles and Permissions**: Determine and assign specific user roles and permissions for the API key.
- **Specify IP whitelist**: Determine the IP addresses that are able to access the API keys.

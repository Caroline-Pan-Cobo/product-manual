sidebarTitle: "为什么要在开发者控制台上注册 API Key ？"
lang: "zh-hans"
description: "了解创建 API Key 的必要性，确保您的应用安全和高效运行。"
---

管理员和操作员可以在开发者控制台上注册 API Key，并分配特定的用户角色、权限和 IP 白名单。根据不同的使用场景，可以注册多个 API Key 。

- **指定用户角色和权限**：为 API Key 确定和分配特定的用户角色和权限。
- **指定 IP 白名单**：确定能够访问 API Key 的 IP 地址。

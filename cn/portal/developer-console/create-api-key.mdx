---

按照以下简单几步即可创建 API key：

1.登录 [Cobo Portal](https://portal.cobo.com/login)。
2.在左侧导航菜单中，点击**开发者** > **API Keys**。
3.点击**创建 API key**。
4.输入 API key 的名称（最多 30 个字符）。
5.输入 API key（64 个字符，只能包含数字 0 至 9 和字母 A 至 F）。
6.从下拉列表中选择用户角色（可选择多个）。
7.选择该 API key 可控制的钱包。
8.勾选两种 key 类型中的一种。
9.输入要列入白名单的 IP 地址（如果要列入白名单的 IP 地址不止一个，则每个 IP 地址之间用逗号隔开）。
10.点击**确认**。

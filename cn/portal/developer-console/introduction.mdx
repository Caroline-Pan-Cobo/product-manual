## 开发者控制台简介

Cobo Portal 的开发者控制台是开发者统一管理 API Key、跟踪 API 日志、监督 Webhook 事件和处理 Callback 消息的平台。这一一站式平台可以帮助开发者控制和优化工作流程。通过该控制台，开发者可实时查看团队的后台部署操作，并提供大量日志用于调试。这可确保开发者拥有必要的工具和信息，以便随时监督和控制团队的运行情况。

## 什么是 Cobo Loop

Cobo Loop 是基于区块链技术的一站式资产转移网络，旨在解决区块链网络中交易拥堵、费用过高等问题。通过提供与传统金融机构类似的清算和结算服务，Cobo Loop 提高了资产转移的效率。Cobo Loop 背后的原理是资金在 Cobo 的全托管钱包（资产钱包）之间流通，实现即时、免费的资产交易。

## 主要特点

- 零交易费：Cobo Loop 中的所有用户都可以即时转账，无需支付任何交易费，使交易更便捷，成本更低。
- 即时确认：交易立即确认，确保为用户提供无缝、高效的体验。
- 多链支持：Cobo Loop 兼容 80 多个区块链网络，为跨平台资产转移提供无与伦比的多功能性。

## Cobo Loop 转账

Cobo Loop 转账是指由 Cobo Loop 网络实现的即时、免费的资产交易，旨在提高区块链生态系统的效率和可访问性。大多数全托管钱包（资产钱包）之间的转账都是 Cobo Loop 转账。如果适用，这些转账会自动即时处理，无需支付任何费用。

您可以通过转账的 UI 检查特定转账是否适用 Cobo Loop 转账（如下图所示）。有关更多详细信息，请参阅[从全托管钱包（资产钱包）转账](/en/portal/transfers/from-asset-wallets)。

<img src="/cn/images/transfers/loop_info_bar.png" className="screenshot_modal" alt="Screenshot of the information bar" />

或者，您可以通过 WaaS 2.0 API 明确强制或禁用特定转账使用 Cobo Loop 转账，具体取决于您的业务需求。如果 Cobo Loop 不适用或被禁用，系统将自动回退到标准的链上转账。有关更多详细信息，请参阅 [Transfer token](https://www.cobo.com/developers/v2/api-references/transactions/transfer-token) 操作中的 [`destination.force_internal`](https://www.cobo.com/developers/v2/api-references/transactions/transfer-token#body-destination-force-internal) 和 [`destination.force_external`](https://www.cobo.com/developers/v2/api-references/transactions/transfer-token#body-destination-force-external) 参数。

## 什么是 Cobo Portal？

<iframe width="560" height="315" src="https://www.youtube.com/embed/5Msjx9WILq8?si=ImrnCF1G_w73FIL7" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>

Cobo Portal 是一个安全高效的数字资产钱包平台。Cobo Portal 凭借其久经考验的安全架构以及对代币和链的全面支持，提供了一系列加密钱包，可满足您的特定需求。此外，Cobo Portal 还提供了各种功能，包括全方位风险控制、对 Web3/DeFi 网络的无缝访问、直观的业务数据概览，以及丰富的 Cobo 自研及第三方开发的 Apps 可供选择。

Cobo Portal 旨在为 Web3 生态系统的各种参与者提供服务：

- DeFi 和质押基金、交易公司、机构投资方和金融服务商：如果您正在寻找安全的数字资产托管解决方案，那么 Cobo Portal 就是您的理想选择，我们支持创建 vault 和钱包，以及直接在平台内进行日常交易。
- 交易所、加密原生公司、OTC 和券商以及零售平台：Cobo 提供钱包即服务 (WaaS) API，让您能够将强大的钱包功能集成到自己的 App 中。利用 Cobo Portal，您可以轻松设置账户、管理 API Key、查看 API 日志，以及使用其他一系列基本功能。
- Web3 开发者：您可以在 Cobo Portal 平台上构建和部署 Apps 作为附加功能。这些 Apps 将为各类数字资产企业提供必要的工具和资源，为此类企业赋能，助力其在 Web3 领发发展。

## 主要特点

### 一站式钱包平台

Cobo Portal 将四种钱包技术整合到单一平台上，提供最先进的安全架构，且其链和代币覆盖范围为业内最广（[点击查看](/cn/portal/supported-tokens) Cobo Portal 支持的代币与链）。

- [全托管钱包](/cn/portal/custodial-wallets/introduction): 全托管钱包采用先进的加密技术和风险控制引擎，确保您的资金免遭未经授权的访问和潜在攻击。它采用了稳健的三层（热-温-冷）私钥存储架构，95% 的资金安全地存储在冷钱包中，只有 5% 的资金存储在热钱包和温钱包中。
- [MPC 钱包](/cn/portal/mpc-wallets/introduction): 多方计算 (MPC) 是一种用于区块链私钥管理的先进加密技术。Cobo Portal 提供了两种类型的 MPC 钱包：机构钱包，让机构能够完全控制自有资金或其终端用户的资产；终端用户钱包，让终端用户能够完全控制其数字资产。
- [智能合约钱包](/cn/portal/smart-contract-wallets/introduction): 这类钱包支持各种智能合约钱包，主要包括 Safe\{Wallet\} 以及其他账户抽象智能钱包。它还支持灵活地委托外部账户 (EOA) 钱包来与智能合约生态系统进行无缝交互。
- [交易所钱包](/cn/portal/exchange-wallets/introduction): 交易所钱包是轻松管理多个交易所账户的一站式解决方案。它将所有交易所账户集中呈现在用户友好的单一界面中，让您能够无缝查看、监控和管理各个交易所中的资产。

### 全方位风险控制
Cobo Portal 提供了各种风险控制机制，旨在解决数字资产存储、管理和转移过程中遇到的挑战，包括[交易风控](/cn/portal/risk-controls/tx-policies-overview)、[业务风控](/cn/portal/risk-controls/governance-policies)以及[用户角色和权限](/cn/portal/risk-controls/roles-and-permissions)。

### Web3/DeFi 访问
全托管钱包（Web3 钱包）、MPC 钱包和智能合约钱包为您提供了安全且可扩展的 Web3 和 DeFi 网关，让您能够向去中心化 App 和金融服务转入资金以及从中转出资金，同时确保资产安全。

### 开发者控制台
Cobo Portal 的开发者控制台为希望控制和优化工作流程的开发者提供了一站式解决方案。您可以利用该控制台完成管理 API Key、跟踪 API 日志、监督 Webhook 事件和处理 Callback 消息等各种操作。例如，日志查看器提供了对整个 API 请求历史记录的深入洞察，并提供有一系列强大的筛选条件，如请求方法、API Key 和状态码。利用这些功能，只需几步操作即可轻松进行调试、故障排除和审计。

### 业务数据概览和报告
Cobo Portal 的[业务数据概览](/cn/portal/dashboard/introduction)是监控和管理数字资产的中心枢纽。它提供了总资产价值、资产分配、交易、定价方案用量包及最新 Cobo 生态系统动态的全面概览。

### Cobo Portal Apps

基于 [WaaS 2.0 API](https://www.cobo.com/developers/v2/guides/overview/introduction)，Cobo 与各类第三方提供商合作，打造了一个强大的商业 App 生态系统 [Cobo Portal Apps](/cn/apps/introduction)，旨在丰富用户数字资产的投资、交易和管理方式。

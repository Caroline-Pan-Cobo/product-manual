## 从 SuperLoop 映射到 BitMart 的资金

SuperLoop 会将您的资金映射到 BitMart 的现货账户（主账户）。一旦入账，您可以在 BitMart 进行现货交易。

## 结算详情

- 结算模式：链下结算
- 结算时间 (UTC+8)：12:00:00

## 使用 SuperLoop 时，BitMart 支持的代币与链

| **代币名称** | **链名称** |
| --------------------------------- | --------------------------------- |
| BTC                            | Bitcoin |
| ETH                            | Ethereum |
| USDC                           | Binance Smart Chain |
| USDT                           | Binance Smart Chain |

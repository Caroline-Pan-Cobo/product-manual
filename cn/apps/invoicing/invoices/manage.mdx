1. 点击 [Cobo Portal](https://portal.cobo.com/login) 上的 <img src="/en/images/common/icon_apps.svg" className="icon" alt="the Apps icon"/>，找到 **Invoicing**，然后点击**打开**，打开 Invoicing。
2. 在**发票**（Invoices）页面上，点击 <img src="/en/images/common/icon_more.svg" className="icon" alt="the overflow icon showing more options"/> 打开更多操作菜单，获取除查看和编辑之外的其他选项。可用操作因发票状态而异。

    | **发票状态** | **可用操作** |
    | ------------------ | ----------------------|
    | 草稿 | 除查看和编辑外，没有其他可用操作 |
    | 发送中/作废中 | 除查看外，没有其他可用操作 |
    | 待付款/逾期 | 发送提醒、作废发票、复制发票 |
    | 已收款/错误/结算中/已结算/结算失败/已作废 | 复制发票 |

    - **编辑**：按需调整发票草稿的详情。有关字段说明，请参见[创建发票](/cn/apps/invoicing/invoices/create)。
    - **发送提醒**（Send Reminder）：打开默认电子邮件客户端，您可以手动向客户发送提醒。
    - **作废发票**（Void Invoice）：提供作废发票的原因。之后可以点击发票列表中**已作废**（Voided）状态旁边的 <img src="/en/images/common/icon_question_circle.svg" className="icon" alt="the question circle icon"/> 查看到该原因。
    - **复制发票**（Duplicate Invoice）：将生成新的发票编号。选择**到期日期**（Payment Due）和**开具日期**（Issued On），其他字段内容将从原发票中复制过来，您可进行修改。

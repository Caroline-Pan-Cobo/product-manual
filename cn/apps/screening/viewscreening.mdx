#### 时间线部分

时间线部分提供一个集中界面，在一个地方管理和跟踪所有合规活动：

- 实时跟踪审核状态变更
- 交易资金状态扫描
- 完整的审计跟踪和操作历史记录

#### 可用操作

根据交易的当前状态，可以使用以下操作：

| 操作 | 效果 |
| :---- | :---- |
| 批准 | • 对于存款：交易通过扫描，资金对用户可用<br/>• 对于提款：交易通过扫描，进入下一阶段 |
| 批准并发出警报 | • 对于存款：交易通过扫描，资金对用户可用，并附加风险交易通知<br/>• 对于提款：交易通过扫描，进入下一阶段，并附加风险交易通知 |
| 拒绝 | • 对于存款：交易未通过扫描<br/>&nbsp;&nbsp;• 目前：交易被标记<br/>&nbsp;&nbsp;• 即将推出：资金将被冻结<br/>&nbsp;&nbsp;&nbsp;&nbsp;• *关于冻结资金的注意事项：如果您使用 [MPC 钱包的自动代币归集功能](/cn/portal/mpc-wallets/ocw/auto-sweep-overview)，冻结资金将不会包含在归集操作中*<br/>• 对于提款：交易未通过扫描，提款被拒绝<br/>• 始终发送风险交易通知 |
| 人工审核<br/><i>（即将推出）</i> | • 对于存款：将启用交易的人工审核<br/>• 对于提款：将启用交易的人工审核 |
| 退回<br/><i>（即将推出）</i> | • 对于存款：将启用从"接收地址"到"发送地址"的资金退回<br/>• 对于提款：不适用 |
| 解冻<br/><i>（即将推出）</i> | • 将启用释放冻结资金，使其对用户可用<br/>• 对于提款：不适用 |

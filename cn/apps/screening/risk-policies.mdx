## 风险处置策略

风险处置策略根据扫描结果自动处理交易。每个策略定义了触发条件（标准）和相应的处理方式。

<Note>
当 Screening 完成交易扫描后：
- 由于[防重複規則](#防重複規則)，每笔交易最多只会匹配一个策略
- 当交易的方向、风险级别和金额范围符合策略的标准时，则触发该策略
- 找到匹配的策略后，Screening 将执行该策略所指定的行动
</Note>

<Warning>如果没有匹配的策略，交易将自动继续进行，**不会进行风险评估**。为确保安全的风险管理，请在使用 Screening 前设置好相应的策略，以涵盖所有可能的交易场景。</Warning>

## 创建、编辑和删除策略

### 创建策略

创建新策略：
1. 进入[风险处置策略界面](#风险处置策略界面)
2. 点击**创建策略**
<img src="/en/images/portal-apps/screening/creating-post-screening-policy.svg" className="screenshot_modal" alt="创建风险处置策略对话框"/>

3. 配置策略的触发条件和处理方式（参见[策略标准](#策略标准)和[策略行动](#策略行动)）
4. 点击**提交**启用策略



#### 防重複規則
系统会防止创建重复的策略。如果新策略与现有的活动策略具有以下所有共同点，则无法创建：
- 相同方向（入币或提币）
- 相同风险级别
- 重复金额范围

例如：
```
策略 1：入币 + 中等风险 + ≥ 500 + < 1000 美元（有效）
策略 2：入币 + 中等风险 + ≥ 400 + < 7000 美元（无效 - 金额范围与策略 1 重复）
策略 2：入币 + 中等风险 + ≥ 600 + < 700 美元（无效 - 金额范围与策略 1 重复）
策略 2：入币 + 中等风险 + ≥ 1000 + < 7000 美元（有效 - 金额范围不重复）
策略 3：提币 + 中等风险 + ≥ 1000 + < 7000 美元（有效 - 不同方向）
```

<Warning>
上述策略示例仅供说明之用。您的实际策略应根据团队的特定合规要求和风险管理策略量身定制。

有关风险评估和策略创建的指导，请参阅 CipherOwl 的[地址风险评分](https://s.cipherowl.ai/Address-Risk-Score-1a0d90dd030f80b5904cd4aa76e15cea)文档，或通过 [CipherOwl](https://www.cipherowl.com/) 网站咨询其合规专家。
</Warning>

### 编辑策略

1. 进入[风险处置策略界面](#风险处置策略界面)
2. 点击要编辑的策略旁边的 <img src="/en/images/common/icon_edit.svg" className="icon" alt="编辑图标"/>
3. 在**编辑风险处置策略**对话框中，根据需要修改策略条件和行动（参见[策略条件](#策略条件)和[策略行动](#策略行动)）
4. 点击**提交**

<Info>编辑策略时同样适用[防重複規則](#防重複規則)。</Info>

### 删除策略

1. 进入[风险处置策略界面](#风险处置策略界面)
2. 点击要删除的策略旁边的 <img src="/en/images/common/icon_delete.svg" className="icon" alt="删除图标"/>
3. 在**删除策略**对话框中，点击**是**确认删除

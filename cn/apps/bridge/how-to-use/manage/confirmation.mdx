本页面帮助用户在最终提交前验证其在[桥接代币](/cn/apps/bridge/how-to-use/manage/bridge-form)中设置的桥接交易详情。

## Token Bridge 预览

### 交易详情
确认页面显示：

1. **钱包**（Wallet）
   - 全托管钱包名称

2. **代币**（Token）
   - 代币符号和金额

3. **从/到**（From / To）
   - 源链（图标 + 名称）
   - 方向箭头
   - 目标链（图标 + 名称）

4. **桥接金额**（Bridge Amount）
   - 带代币符号的金额
   - 等值法币金额

5. **预计时间**（Estimated Time）
   - 预计完成时间

6. **交易费用**（Transaction Fee）
   - Token Bridge 交易费用

7. **收款金额**（Amount to Receive）
   - 显著显示的最终金额
   - 代币符号（目标代币，例如 BTC→WBTC 中的 WBTC）
   - 等值法币金额

## 交易流程

1. **验证**
   - 验证所有交易细节
   - 检查费用和预估收款金额
   - 确认桥接路径正确

2. **操作**
   - 点击"取消"返回代币桥接
   - 点击"提交"继续交易

3. **确认**
   - 提交后显示确认消息：
     - 标题："桥接请求已提交"
     - 消息："您可以在**活动**（Activity）部分追踪状态"
   - 选项：
     - "关闭"：关闭确认

4. **提交后**
   - 交易出现在**活动**中。
   - 需要 Cobo Guard 审批。
   - 交易详情中可查看状态更新。
